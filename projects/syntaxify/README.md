# Syntaxify

Easy to use syntax highlighting for Angular. Currently supports the following languages:

- TypeScript
- JavaScript
- HTML
- C++

## Installation

```
npm i syntaxify
# or
yarn add syntaxify
```

## Usage

```
import { SyntaxifyModule } from 'syntaxify';

@NgModule({
  imports: [
    SyntaxifyModule
  ]
})
```
